# Kreeda-Ankana (Sports)

Kreeda-Ankana is an Android application developed using Kotlin and Firebase to organize sports activities in rural and semi-urban areas. The app helps users book sports ground slots, participate in match challenges, manage scores, and improve sports coordination digitally.

## Features

* User Registration and Login using Firebase Authentication
* Ground Slot Booking System
* Challenge Board for Teams
* Match Management
* Score Tracking
* Dashboard Navigation
* Firebase Realtime Database Integration
* Simple and User-Friendly UI

## Tech Stack

* Kotlin
* Android Studio
* Firebase Authentication
* Firebase Realtime Database
* XML Layout Design

## Objective

The main goal of this project is to provide a structured digital platform for organizing local sports activities, reducing conflicts in ground usage, and encouraging community participation through technology.
